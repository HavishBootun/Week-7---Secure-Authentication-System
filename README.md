## Description

This project implements a secure, command-line authentication system using Python and bcrypt hashing.  
It forms part of Week 7 Lab activities for CST1510.

Users can:
- Register a new account
- Log in using their credentials
- Have their passwords securely hashed (never stored as plaintext)
- Enjoy simple input validation and secure storage

---

## Features

- bcrypt password hashing with salt
- Secure password verification
- Duplicate username prevention
- File-based user storage (`users.txt`)
- Validation for usernames and passwords
- Fully interactive command-line menu
